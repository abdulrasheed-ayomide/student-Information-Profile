# Student Information Management System

This is a responsive Student Information Management System built with **HTML, CSS, and JavaScript**.

## Features
- Add student name, gender, and phone number
- Search students by name or phone number
- Delete student records
- Data persistence using browser **localStorage**
- Fully responsive layout (desktop & mobile)
- Clean and professional UI design
- No external libraries or frameworks

## Technologies Used
- HTML5
- CSS3 (Flexbox & Media Queries)
- JavaScript (DOM Manipulation & localStorage)

## How It Works
Student records are stored in the browser using localStorage, allowing data to persist even after page refresh or browser restart.

## Responsive Design
The layout adapts seamlessly to different screen sizes while maintaining a structured table layout without text overflow or collisions.

## Usage
1. Enter student details
2. Click **Add Student**
3. Search or delete records as needed
4. Data remains saved automatically
